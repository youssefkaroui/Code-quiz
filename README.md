## Purpose

Code-quiz is a web-application that lets the user take an evalution quiz on Java-Script. The quiz is timed and it deducts time every time the user gets the answer wrong. It then gives the user a score at the end of the quiz. After that it does store those score and sorts them from high to low. 


## Installation 

Launch the website on your default browser
